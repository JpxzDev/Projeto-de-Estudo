Aula 1 - Curso Basico HTML

O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).
O HTML é a linguagem responsável pela estrutura das páginas da web.
Com ele, podemos criar títulos, textos, imagens, links, listas e muito mais. Todo site na internet utiliza HTML.

Estrutura Básica de uma Página HTML

documento HTML possui uma estrutura padrão: Todo documento HTML possui uma estrutura padrão:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
</head>
<body>

</body>
</html>

Explicação:

<!DOCTYPE html> → Informa ao navegador que o documento utiliza HTML5.
<html> → Contém todo o conteúdo da página.
<head> → Guarda informações importantes da página.
<meta charset="UTF-8"> → Permite utilizar caracteres especiais, como acentos.
<title> → Define o nome que aparecerá na aba do navegador.
<body> → Contém tudo o que será exibido ao usuário.

Títulos

        Para criar títulos utilizamos as tags de <h1> até <h6>.
<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Título Menor</h3>

Parágrafos

        Para adicionar textos comuns usamos a tag <p>.

<p>Este é um parágrafo.</p>

Quebra de Linha

Para quebrar uma linha utilizamos a tag <br>.

<p>Olá!<br>Bem-vindo ao curso de HTML.</p>

Imagens

        As imagens são adicionadas com a tag <img>.

<img src="imagem.jpg" alt="Descrição da imagem">,

src → Caminho da imagem.
alt → Texto alternativo da imagem.

Links

        Para criar links utilizamos a tag <a>.

<a href="https://www.google.com">Visitar o Google</a>

Lista

        Lista não ordenada:

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>
     
         Lista ordenada:


<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
    <li>Terceiro passo</li>
</ol>

Divs

        A tag <div> é usada para organizar e
 separar partes da página.

<div>
    <h2>Meu Card</h2>
    <p>Este é um exemplo de conteúdo dentro de uma div.</p>
</div>
